FAQ ヱビスポップのよくある質問

予約なしでも入れますか？

はい、空席があれば予約なしでも大歓迎です！ただし、金曜日・土曜日などの週末は大変混み合いますので、スムーズなご案内のためにお電話かWEBからの事前ご予約をおすすめしております。

お酒があまり強くないのですが、楽しめますか？

もちろんです！当店自慢のゴロゴロ生搾りサワーは、すべてノンアルコール（ソーダ割り）に変更可能です。その他ソフトドリンクも充実しておりますので、安心してお越しください！

クレジットカードや電子マネーは使えますか？

はい、各種クレジットカード（Visa, Master, JCB, AMEX等）、PayPayやd払いなどのQRコード決済、交通系電子マネーがすべてご利用いただけます。
