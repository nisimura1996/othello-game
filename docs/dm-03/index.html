瀬戸内の旬を、職人仕込みの生パスタで。

Concept

松山が誇る「富士麺ず工房」の生麺を使用。もちもちの食感と、シェフこだわりのソースが織りなす至福の一皿をご提供いたします。

今治の漁師・藤本さんから届く鮮魚など、瀬戸内の恵みをふんだんに取り入れ、イタリアから直輸入した美しい食器に彩を添えて。

日常の贅沢に、大切な記念日に。心がほどける豊かなひとときを、Osteria Splendeでお過ごしください。

こだわりの生パスタ

厳選された小麦と熟成技術で作られる富士麺ず工房の生麺。ソースとの相性を極限まで追求しました。

瀬戸内の恵み

今治の漁師から直接仕入れるイワシや真鯛など、地元ならではの鮮度と旨さを一皿に凝縮します。

至福の空間

イタリア中から集めた鮮やかな食器と、レンガ造りの落ち着いた店内。五感すべてで愉しむ時間を。

06

よくある質問

予約なしでも入れますか？

はい、空席がある場合はご案内可能ですが、お席に限りがございますので、特に週末のディナーや大人数でのご利用の際は、事前のご予約をお勧めしております。

アレルギーへの対応は可能ですか？

可能な限り対応させていただきます。ご予約時に苦手な食材やアレルギーについてお伝えいただければ、シェフが最適なメニューをご提案いたします。

個室はありますか？

完全な個室はございませんが、店奥に半個室のように落ち着いて過ごせるボックス席をご用意しております。会食や記念日等にご活用ください。
