店内紹介

洗練された大人のための空間

外観

松山市の中心部に佇む、風情溢れる木と石板を基調とした上質なエントランス。鈍く光る「極」の真鍮看板が、お客様を非日常の美食体験へと誘います。

内観（空間）

伝統的な和の意匠とモダンな美学が美しく融合した、極めてシックな漆黒の空間。間接照明の柔らかな光が揺らぎ、大人の静寂な夜を豊かに引き立てます。

カウンター席

樹樹数十年の国産檜の一枚板を使用した極上のカウンター席。目の前で繰り広げられる職人の繊細な包丁さばきや、肉が網の上で焼かれる炭火のシズル感を五感すべてでご堪能いただけます。

完全個室（和室・洋室）

プライベートを完全に守る上質な格子戸付きの個室空間。ビジネスの大切な接待や、ご家族の長寿のお祝い、ハレの日のデートなど、様々な特別なシーンで周囲を気にせずゆったりとお愉しみいただけます。

中庭（日本庭園）

四季折々の表情を見せる静かな中庭（日本庭園）。格式高い石灯籠と手入れの行き届いた植栽が、個室の窓越しに優美に広がり、極上の美食体験をより一層風情豊かに彩ります。

よくある質問

ご来店にあたっての疑問に誠実にお答えします

焼肉と和食を同時に注文することは可能ですか？

はい、可能です。当店は焼肉と和食の両方を同一のお席でお愉しみいただけるユニークなスタイルを採用しております。お連れ様とお好きなメニューをそれぞれお選びいただけます。

食物アレルギーや苦手な食材への対応は可能ですか？

はい、可能な限り柔軟に対応させていただきます。出汁をベースとした和食料理が多いため、特定の食材やアレルギーがある場合は、ご予約の際に事前にお申し付けください。職人が特別に仕立てます。

個室の指定や席の希望は聞いてもらえますか？

はい、個室のご指定や、庭園を望むお席などのご要望も承っております。ただし、お部屋数に限りがございますので、大切な日のご予約はできるだけお早めにご連絡いただけますと幸いです。

スマートフォンの決済やクレジットカードは使えますか？

はい、現金のほか、主要なクレジットカード（Visa, Master, JCB, AMEX, Diners）や、各種QRコード決済（PayPayなど）をご利用いただけます。
