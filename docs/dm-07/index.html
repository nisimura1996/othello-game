FAQよくあるご質問

ビールがあまり詳しくないのですが、おすすめを選んでもらえますか？

もちろん喜んでお選びいたします！当店のスタッフはクラフトビールが大好きで、知識も豊富です。「すっきり系」「フルーティー」「苦め」など、お客様のお好みやご注文の料理に合わせて最適な一杯をご提案いたします。お気軽にスタッフへお声がけください。

予約なしの当日利用は可能ですか？

はい、大歓迎です。ただし、週末や時間帯によってはテーブル席およびカウンター席が満席になる可能性がございますので、確実にご案内できるよう、事前のお電話またはWeb予約をおすすめしております。

貸切や大人数での宴会は対応していますか？

はい、対応しております。着席で最大25名様、立食で最大40名様までの貸切パーティーを承っております。ご予算やご要望に応じたカスタムプランもご提案可能ですので、お気軽にお電話にてお問い合わせください。

支払方法は何がありますか？

現金のほか、各種クレジットカード（Visa, Master, JCB, AMEX等）やPayPayなどのQRコード決済をご利用いただけます。

駐車場はありますか？

恐れ入りますが、当店専用の駐車場はございません。近隣のコインパーキングをご利用ください。3,000円以上のお会計で、提携パーキングの1時間無料駐車券をお渡ししております。
